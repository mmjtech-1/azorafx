# Azora FX — Professional Trading Platform

## Setup

1. Clone repo
2. `cp .env.local.example .env.local`
3. Fill in all environment variables
4. `pnpm install`
5. Run Supabase migrations from `supabase/migrations/` in order `001-012`
6. `pnpm dev`

## Deploy

1. Push to GitHub
2. Connect to Vercel
3. Add environment variables in Vercel dashboard
4. Deploy

## Tech Stack

Next.js 14, Supabase, Claude AI, PayPal, Tailwind CSS, Framer Motion
